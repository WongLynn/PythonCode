<h5 id = 'c2'>2. 项目说明</h5>

一个Mp3播放器(仅逻辑实现)

---------------

1. 歌曲本身存在硬盘，路径存储在MySQL数据库

2. 终端界面实现控制歌曲的播放，暂定，继续操作

3. 终端界面实现歌曲在数据库的存储，删除操作

---------------
